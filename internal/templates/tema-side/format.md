# Tema Side Format

Vi trenger et format for å presentere undervisngings tema på.

Dette er noe som ikke blir endelig avgjort her, men vi trenger noe som fungerer til det formelle er på plass.

## Innholds Krav

- Strukturert Metadata
  - synlighet: intern | åpen
  - livsyklus: utkast | høring | akseptert
- Bakenforliggende Teori
  - Kort fortalt, lenk til autorative kilder
  - Tegninger/Kart for å illustrer relasjoner
- Konkretisering
  - Hva, hvor, når blir det brukt?
  - Lenke til eksempler
    - Git repository fra undervisning
    - Ekstern eksempler
- Videre lesning
- Referanse Liste
